## Submodules

This repository includes the following submodules:

- [pfun-cma-model](docs/pfun-cma-model/): Documentation for the PFun CMA Model

## Building Documentation

To build the documentation for the pfun-cma-model submodule:

```bash
./scripts/build-pfun-cma-docs.sh
```

This will build the documentation and make it available at `/pfun-cma-model/` on your local server.

## Updating Submodules

To update the submodules to the latest version:

```bash
git submodule update --remote
```
